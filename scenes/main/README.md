# Main Scene

`Main.tscn` is the Phase 1 startup scene. It defines only the stable root skeleton:

- `GameRoot` groups the playable run tree.
- `Managers` is reserved for future manager nodes.
- `WorldRoot` is reserved for map and world visuals.
- `UIRoot` is reserved for Control-based UI scenes.

No gameplay rules, manager logic, map rendering, or UI systems live here yet.
